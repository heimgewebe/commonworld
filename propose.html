What happens next

The suggestion is not published automatically. Commonworld reviews identity, sources, Commons characteristics, ways to engage, location precision, privacy and freshness. Only a separately reviewed repository commit can publish a catalog entry.

Public visibility: The preferred intake path is a public GitHub issue. Do not enter an email address, phone number, private address, coordinates, apartment, rooftop, router or household information. Your GitHub account serves as the contact path; Commonworld collects no additional contact field.

Without GitHub, or if the network fails, you can download a validated JSON file locally. Commonworld does not store form input.
