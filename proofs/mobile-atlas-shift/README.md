# Mobile Atlas Shift Proof

This proof visualizes the mobile-first `Karte -> Aether` interaction model.

It is intentionally static: no backend, no accounts, no public submissions and no weltgewebe write path.

## Concept

- `Karte`: a map-like projection for local commons.
- `Aether`: thematic swipe streams for digital and hidden commons.

Horizont is not a selectable mode. It is the transition behavior of hybrid commons while shifting between Karte and Aether.

Hybrid commons can appear in both projections:

- on the map as a local marker with a rising code pillar;
- in the Aether as a digital card with an Ortssignal.

Project profile remains a focus state, not a main navigation pillar.
