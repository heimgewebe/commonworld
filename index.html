commonworld is the public Commons atlas entry.

This static hub connects the current proof surfaces without creating a backend, public submissions, user accounts or a weltgewebe write path. It now separates visual explanation, location rendering, projection focus and static data-quality checks so a reader can see what each proof actually proves.
